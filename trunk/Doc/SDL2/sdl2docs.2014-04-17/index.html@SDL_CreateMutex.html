SDL_CreateMutex

Use this function to create a new mutex.

Contents

  1. SDL_CreateMutex
    1. Syntax
    2. Return Value
    3. Code Examples
    4. Remarks
    5. Related Functions

Syntax

SDL_mutex* SDL_CreateMutex(void)

Return Value

Returns the initialized and unlocked mutex or NULL on failure; call SDL_GetError() for more information.

Code Examples

SDL_mutex *mutex;

mutex = SDL_CreateMutex();
if (!mutex) {
  fprintf(stderr, "Couldn't create mutex");
  return;
}

if (SDL_LockMutex(mutex) == 0) {
  /* Do stuff while mutex is locked */
  SDL_UnlockMutex(mutex);
} else {
  fprintf(stderr, "Couldn't lock mutex");
}

SDL_DestroyMutex(mutex);
