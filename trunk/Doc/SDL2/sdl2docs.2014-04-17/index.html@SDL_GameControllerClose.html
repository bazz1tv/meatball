SDL_GameControllerClose

Use this function to close a game controller previously opened with SDL_GameControllerOpen().

Contents

  1. SDL_GameControllerClose
    1. Syntax
    2. Function Parameters
    3. Code Examples
    4. Remarks
    5. Related Functions

Syntax

void SDL_GameControllerClose(SDL_GameController* gamecontroller)

Function Parameters

gamecontroller

a game controller identifier previously returned by SDL_GameControllerOpen()

Code Examples

/* Attempt to open every controller. */
for (int i = 0; i < SDL_NumJoysticks(); ++i) {
    if (SDL_IsGameController(i)) {
        SDL_GameController *controller = SDL_GameControllerOpen(i);
        if (controller) {
            SDL_GameControllerClose(controller);
        } else {
            fprintf(stderr, "Could not open gamecontroller %i: %s", i, SDL_GetError());
        }
    }
}
